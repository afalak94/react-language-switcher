# Vanilla CSS

While a lot new patterns such as CSS-in-JS or CSS Modules have appeared, vanilla CSS is still probably the most widely used solution.

You can easily style `<Switcher />` component by overriding the optional **className** props.

## Preview

<Preview withSource="none">
  <SwitcherProvider>
    <LanguageSwitcher
      languageList={['en', 'de', 'fr']}
      containerClassName="language-switcher"
      listClassName="language-switcher__list"
      itemClassName="language-switcher__item"
      buttonClassName="language-switcher__button"
      buttonActiveClassName="language-switcher__button--active"
    />
  </SwitcherProvider>
</Preview>

## Usage

```tsx
<SwitcherProvider>
  <LanguageSwitcher
    languageList={['en', 'de', 'fr']}
    containerClassName="language-switcher"
    listClassName="language-switcher__list"
    itemClassName="language-switcher__item"
    buttonClassName="language-switcher__button"
    buttonActiveClassName="language-switcher__button--active"
  />
</SwitcherProvider>
```

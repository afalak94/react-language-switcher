# LanguageProvider

A Provider component, that provides `<LanguageContext />` to its children.

It allows any child, regardless of the depth, to have access to the context value object based on LanguageContextProps

## Purpose

## Element

```tsx
<LanguageProvider />
```

## Usage

